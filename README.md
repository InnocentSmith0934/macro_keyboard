# macro_keyboard

Code for a macro keyboard using a Teensy and a six-key keyswitch sample set.

WASD keyboard switch sampler:
http://www.wasdkeyboards.com/index.php/products/sampler-kit/wasd-6-key-cherry-mx-switch-tester.html

Teensy board:
https://www.pjrc.com/teensy/index.html

Bounce library (this code uses Bounce version 1):
https://github.com/thomasfredericks/Bounce2

Arduino keyboard libary reference:
https://www.arduino.cc/reference/en/language/functions/usb/keyboard/

Based on the following Reddit post, imgur thread, and pastebin text:
https://www.reddit.com/r/MechanicalKeyboards/comments/2yb568/modification_turned_my_wasd_keyboards_switch/
